App Mobile Data Mahasiswa

Project ini adalah aplikasi mobile yang digunakan untuk mengelola data mahasiswa dengan sistem CRUD (Create, Read, Update, Delete). Fungsinya antara lain:
Create → menambahkan data mahasiswa baru.
Read → menampilkan daftar data mahasiswa.
Update → mengedit data mahasiswa yang sudah ada.
Delete → menghapus data mahasiswa.
Tujuan aplikasi ini adalah agar pengelolaan data mahasiswa menjadi lebih terstruktur, efisien, dan mudah diakses kapan saja melalui perangkat mobile, baik untuk pihak kampus maupun admin pengelola
